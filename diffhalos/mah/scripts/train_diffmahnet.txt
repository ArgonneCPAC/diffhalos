"Scripts to train diffmahnet using float64, modifying Alan's original scripts"

- Finished v2 models. Trained using v1 as the initial model, and performing kdescent fitting to the 4D space {log(M(t)), t, log(M_obs), t_obs}.
    - Model names:
        cenflow_v2_0_float64.eqx
        satflow_v2_0_float64.eqx
    - Respective creation scripts:
        mpiexec -n 1 $watchmemory python -W ignore ~/local/diffhalos/diffmahpop/scripts/kdescent_train_float64.py cenflow_v2_0_float64.eqx --num-kernels 20 --num-fourier-kernels 20 --plot-loss-curve --seed 2 --include-test --initial-model cenflow_v1_0-kde_20_20train.eqx --steps 1200 --learning-rate 3e-5

        mpiexec -n 1 $watchmemory python -W ignore ~/local/diffhalos/diffmahpop/scripts/kdescent_train_float64.py satflow_v2_0_float64.eqx --num-kernels 20 --num-fourier-kernels 20 --plot-loss-curve --seed 2 --include-test --initial-model satflow_v1_0-kde_20_20train.eqx --steps 1200 --learning-rate 3e-5 --sats
